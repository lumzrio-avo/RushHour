# 3D Rush Hour 华容道解谜游戏 - 个人算法模块

## 项目简介
本项目为团队课程项目，基于 Python + Ursina 引擎开发的 2D/3D 混合版汽车华容道解谜游戏。
本仓库仅包含我个人负责的核心模块。

## 个人贡献声明
- 算法层：独立实现 BFS 广度优先搜索算法，求解理论最优步数，支撑智能提示与星级评分机制
- 逻辑层：独立实现小车移动、碰撞检测、胜利判定、步数撤销与重置等核心游戏逻辑
- 关卡设计：独立完成多梯度关卡难度设计与数据配置

完整项目由团队负责人维护，我的角色为开发成员（算法模块负责人）。

## 代码运行说明
本仓库包含个人负责的算法与逻辑模块，无法独立运行完整游戏。
如需查看完整项目效果，请联系我获取团队仓库链接。

## 文件结构
- bfs_solver.py      # BFS 最优解求解器
- game_logic.py       # 移动、碰撞检测、胜利判定
- undo_manager.py     # 撤销与重置功能
- levels.json         # 关卡配置数据
- main.py             # 最小启动示例（棋盘渲染+移动交互）
